# superorder

## 安裝及使用

### python env
```
$ pyvenv projectenv
$ source projectenv/bin/activate
```

### 安裝
```
~/$ git clone https://github.com/tim124058/superorder.git
~/$ cd superorder
~/superorder$ pip3 install -r requirements.txt
```

### 開啟、關閉 server
```
~/superorder$ python3 manage.py runserver
```
開啟瀏覽器網址打上：<http://127.0.0.1:8000/>

關閉sever : `ctrl+c`
